Le Homebrew France > PS2-Guide > FreeDVDBoot

Rejoignez Le Homebrew France pour obtenir de l'assistance ou nous aider à développer le guide.

N'oubliez pas de lire l'intégralité des pages avant de procéder pour ne pas prendre le risque de manquer une étape.

Prérequis

  • L'ISO de la dernière version de FreeDVDBoot.
    1. ISO pour les versions 3.04
    2. ISO pour toutes les PS2 Slim
    3. ISO pour les versions 2.10 et 2.12
  • La dernière version du logiciel ImgBurn.
  • La dernière version de l'installateur FreeMCBoot.
  • Une clé USB.
  • Une carte mémoire PlayStation 2 (de préférence officielle, elle contient la mention "MagicGate")
  • Un DVD-R (de préférence de marque Verbatim).

	
I - Préparation de la clé USB

	
    
		
  • Insérez la clé USB sur un PC.
    • 
		
  • Formatez la clé USB en FAT32, si vous ne savez pas comment faire, cliquez ici.
    • 
		
  • Ouvrez l'archive  FMCB-1966.7z , ouvrez ensuite le dossier  FMCBinst-1966-[2023-02-17] 
		
  • Copiez les dossiers  INSTALL  et  LANG  à la racine de la carte SD (la racine signifie que vous n'êtes dans aucun dossier).
    • 
	

	

	
II - Graver le DVD

	
    
		
  • Démarrer le logiciel ImgBurn.
    • 
		
		
  • Sélectionner "Write Image File To Disc".
    • 
		
	

	
    
		
  • Cliquez sur l'icône de dossier sous source .
    • 
		
  • Sélectionnez l'ISO FreeDVDBoot selon votre modèle de console dans la section "Prérequis".
    • 
		
  • Sélectionnez votre graveur de CD/DVD.
    • 
		
  • Insérez un DVD-R.
    • 
		
  • Ajoutez la vitesse la plus basse possible de votre lecteur CD/DVD
    • 
		
  • Sélectionnez l'option "Write" en bas à gauche de la fenêtre.
    • 
		
  • Attendre la fin du processus de gravure.
    • 
	

	

	
III - Installation de FreeMCBoot (FMCB)

	
Nous allons maintenant procéder à l'installation du programme "FreeMCBoot" sur votre carte mémoire.

	
    
		
  • Insérez votre carte mémoire dans la fente 1 de votre PlayStation 2 ainsi que votre clé USB.
    • 
		
  • Insérez le DVD-R que vous avez précédemment gravé dans le lecteur DVD de votre PlayStation 2.
    • 
		
  • Autorisez la console à démarrer sur le DVD.
    • 
	

	
    
		
  • Votre console devrait lancer le logiciel "uLaunchELF".
    • 
	

	
    
		
  • A partir de là, sélectionnez "File Browser" en appuyant sur .
    • 
		
	

	
    
		
  • Sélectionnez  mass:/  en appuyant sur .
    • 
	

	
    
		
		
  • Sélectionnez le fichier  fmcbinstaller.elf .
    • 
	

	
    
		
  • Le programme FreeMCBoot Installer devrait démarrer.
    • 
	

	
    
		
  • Utilisez  et  pour naviguer jusqu'au menu "Utilitaire carte mémoire".
    • 
	

	

		
La prochaine étape effacera toutes les données de votre carte mémoire ! Veillez à faire une sauvegarde de cette dernière !

	

	
    
		
  • Cliquez sur "Formater CM" avec le bouton .
    • 
		
		
  • Utilisez  et  pour naviguer jusqu'au menu "Confirmation" et sélectionnez "Normale".
    • 
		
		
  • Autorisez maintenant l'installation à se poursuivre.
    • 
	

	

		
Vous venez à présent d'installer FreeMCBoot sur votre carte mémoire. Vous pouvez poursuivre vers fin de l'installation.
